# Dasappan

A Flutter application for streaming music from YouTube Music.

## Features

- Play (almost) any song or video from YouTube Music.
- High quality audio.
- Ad free.
- No subscription.
- Background playback.
- Dark,Light theme based on System settings.
- Mini Player.
- Voice search.
- Toggle audio quality.
- Reorderable Queue.
- Sleep Timer
- Search for songs and playlists.
- Neat and clean code.
- App close bug fixed.
