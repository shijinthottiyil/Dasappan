# Dasappan

A Flutter application for streaming music from YouTube Music.


## Features

- Play (almost) any song or video from YouTube Music.
- High quality audio.
- Ad free.
- Background playback.
- Simple and elegant UI.
- Search for songs and playlists.
- Neet and clean code.
