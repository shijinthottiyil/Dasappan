# Dasappan

A Flutter application for streaming music from YouTube Music.


## Features

- Play (almost) any song or video from YouTube Music.
- High quality audio.
- Ad free.
- No subscription.
- Background playback.
- Glass Morphic Ui.
- Mini Player.
- Voice search.
- Toggle audio quality.
- Reorderable Queue.
- Dynamic wallpaper from Unsplash api.
- Search for songs and playlists.
- Neet and clean code.
